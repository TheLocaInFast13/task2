Malédiction de Disparition

Type : Enchantement Maudit

Disponibilité : Rare (portails du Nether)

Effet : Fait disparaître le joueur lorsqu'il meurt avec un objet maudit.

Obtention

Ce livre enchanté est parfois trouvé dans des coffres de portails ruinés dans le Nether.

Effets

  • Lorsque le joueur meurt en portant un objet avec cet enchantement, son corps disparaît instantanément.
  • Les autres joueurs ne peuvent pas voir son loot ni le récupérer.
  • Effet irréversible, même après réapparition.

Trivia

Certains joueurs pensent que cet enchantement est une relique du Nether, utilisée autrefois par les Piglins pour emprisonner les âmes.
